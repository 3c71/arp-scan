Checks to perform on all supported platforms before release:

1.  Build and check for no warnings or errors: ./configure; make; make install

2.  make check

3.  Single target test: arp-scan <IP>

4.  Localnet test: arp-scan [-I <if>] --localnet

5.  Range test: arp-scan [-I <if>] a.b.c.d/x

6.  get-oui/get-iab test

7.  Fingerprint test: arp-fingerprint <IP>

8.  manpage test: read manpages, and check for correct formatting.
